<h2>Never Let Me Go</h2>
<p class="ratd"> Rated <em>M</em></p>
<p><em>Review by David Stratton</em></p>
<p>The story of NEVER LET ME GO unfolds in a parallel universe where an unspecified scientific breakthrough in 1952 has, by 1967, brought about a life expectancy of over 100. This has come about thanks to a National Donor Programme in which boys and girls are raised specifically to become organ donors servicing the rest of society. <br/><br/>Kathy, ISOBEL MEIKLE-SMALL, Ruth, ELLA PURNELL and Tommy, CHARLIE ROWE, meet at Hailsham boarding school and are at first too young to understand the fate that will await them. One of the teachers, Miss Lucy, SALLY HAWKINS, tries to prepare them...<br/><br/>Several years later the trio, now adults and played by Carey Mulligan, Keira Knightley and Andrew Garfield, are staying at a sort of half way house called The Cottages - they will soon be donors, but meanwhile Ruth and Tommy have become lovers to the quiet distress of Kathy...<br/><br/>Alex Garland's adaptation of the novel by Kazuo Ishiguro, author of THE REMAINS OF THE DAY, makes some major structural changes, revealing much earlier the reason why the story's protagonists are being brought up in this strange fashion. Ishigoro has approved the changes, but the film still demands much of audiences, requiring a suspension of belief that I found it hard to cope with. The main problem is that we know nothing about the real world, the world where the people who receive these organ donations live their lives. <br/><br/>The result is that the story is taking place in a vacuum. As a triangular love story, this doesn't really soar either, despite good performances from a hip young cast, and the end result is that a story that should have been terribly affecting, isn't. <br/><br/>It's beautifully filmed, meticulously directed by Mark Romanek, but ultimately inert.</p>
<h2>Further comments</h2>
<p>DAVID: Margaret?<br/><br/>MARGARET: Isn't it strange? I agree with you. And when I saw this I thought that what was on the page maybe doesn't translate so well to the screen. It's obvious that he is trying to deal with all of us. Our mortal condition, and I think it's about the importance of love being recognised within whatever earthly span we have. But to create this world and particularly set it in the 1950s, I don't know why it wasn't set in some futuristic world where it would have had a greater believability. Setting in the 1950s - transplants weren't even in until, what 1960, '61 or whatever<br/><br/>DAVID: Yes.<br/><br/>MARGARET: And...<br/><br/>DAVID: Well, it's suggesting that the breakthrough came much earlier, hence this...<br/><br/>MARGARET: Is that why we are living longer?<br/><br/>DAVID: Maybe there are donors out there that we don't know about in this parallel universe.<br/><br/>MARGARET: I don't know but it strikes me that there is an enormous reach with this film and actually, there's a lot of talent: Ishigoro's novel...<br/><br/>DAVID: Oh, yes.<br/><br/>MARGARET: ...Alex Garland's screenplay. They're friends, apparently, and have worked together, really a lot and you so want it to be grand and meaningful and somehow for me it wasn't.<br/><br/>DAVID: It doesn't soar, does it?<br/><br/>MARGARET: No.<br/><br/>DAVID: I really stays surprisingly prosaic, given it's such an unusual subject.<br/><br/>MARGARET: And England in the 1950s, so brown and grey.<br/><br/>DAVID: Oh, come on, I grew up in England in the 1950s.<br/><br/>MARGARET: I know. That's what .<br/><br/>DAVID: It was that brown and grey?<br/><br/>MARGARET: Yes, it was. It was always brown and grey.<br/><br/>DAVID: No, it wasn't. What are you giving this?<br/><br/>MARGARET: I'm giving this three.<br/><br/>DAVID: Me too: three.</p>

function cnt(theform){
	var y=theform;
	var r = 0;
	a=y.replace('\n',' ');
	a=a.split(' ');
	for (z=0; z<a.length; z++) {if (a[z].length > 0) r++;}
	return r;
}

// generic get by class name
function getElementsByClassName(strClass, strTag, objContElm) {
  strTag = strTag || "*";
  objContElm = objContElm || document;    
  var objColl = objContElm.getElementsByTagName(strTag);
  if (!objColl.length &&  strTag == "*" &&  objContElm.all) objColl = objContElm.all;
  var arr = new Array();                              
  var delim = strClass.indexOf('|') != -1  ? '|' : ' ';   
  var arrClass = strClass.split(delim);    
  for (var i = 0, j = objColl.length; i < j; i++) {                         
    var arrObjClass = objColl[i].className.split(' ');   
    if (delim == ' ' && arrClass.length > arrObjClass.length) continue;
    var c = 0;
    comparisonLoop:
    for (var k = 0, l = arrObjClass.length; k < l; k++) {
      for (var m = 0, n = arrClass.length; m < n; m++) {
        if (arrClass[m] == arrObjClass[k]) c++;
        if ((delim == '|' && c == 1) || (delim == ' ' && c == arrClass.length)) {
          arr.push(objColl[i]); 
          break comparisonLoop;
        }
      }
    }
  }
  return arr; 
}
